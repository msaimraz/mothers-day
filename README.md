## 🌸 About

This is a beautifully crafted, emotion-driven landing page built to honor and celebrate **every mother** — the ones who sacrifice silently, love unconditionally, and give everything without ever expecting a thank you.

> *"Life doesn't come with a manual, it comes with a mother."*

---

## ✨ Features & Animations

| Feature | Description |
|---------|-------------|
| 🌺 **Falling Petals** | Continuous rain of animated flower petals across the screen |
| 💖 **Sparkle Cursor** | Magical sparkle trail that follows your mouse |
| 🦋 **Floating Butterflies** | Two butterflies gracefully drifting across the viewport |
| 🎠 **Parallax Hero** | Hero section with depth-based scroll parallax effect |
| 📸 **Image Gallery** | 3-card hover gallery with zoom & overlay transitions |
| 💬 **Quote Carousel** | Auto-rotating carousel with 4 inspirational quotes |
| 🏆 **Tribute Cards** | 6 animated cards celebrating what makes mom special |
| 📅 **Animated Timeline** | Scroll-triggered timeline of a mother's journey |
| 💌 **Love Letter** | A heartfelt open letter to every mother |
| 📊 **Stats Counter** | Fun counter section with motherhood stats |
| 🖼️ **Infinite Photo Strip** | Auto-scrolling horizontal photo carousel |
| 📝 **Message Wall** | Pinboard-style notes from children around the world |
| 🎯 **Scroll Animations** | Every section reveals with smooth scroll-triggered transitions |
| 💫 **Heartbeat Emojis** | Pulsing heart/flower animations throughout |

## 🛠️ Tech Stack

```
📄 HTML5        → Semantic structure
🎨 CSS3         → Animations, gradients, transitions, keyframes
⚡ JavaScript   → Intersection Observer, DOM manipulation, carousels
🖼️ Unsplash     → Free high-quality stock images
🔤 Google Fonts → Playfair Display, Cormorant Garamond, Dancing Script, Lora
```

**Zero dependencies. Zero frameworks. Just pure code and love.**

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/msaimraz/mothers-day.git
cd mothers-day
```

### Run locally

Simply open `index.html` in your browser — no build step required!

```bash
# Using VS Code Live Server
# Or simply:
open index.html
```

### Deploy to Vercel

```bash
npm i -g vercel
vercel
```
